Sprite image generator. PHP methods are available in order to get positioning of each element and write CSS rules.

== Description ==
Nusprite plugin allows you to generate sprite images from featured images of posts and get CSS rules with PHP.

== Features ==
- Choice in image source (feature image or custom field)
- Versioning system for long term file caching

nusprite.pot file available

== Installation ==
1. Upload the full directory into your '/wp-content/plugins' directory
2. Activate the plugin at the plugin administration page

You will find 'Nusprite' menu in your WordPress admin panel

== Screenshots ==
1. Add feature images
2. Generate sprite
3. Get CSS rule with PHP
